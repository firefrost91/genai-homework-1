# GenAI-homework-1
Repository Link: https://github.com/firefrost91/genai-homework-1

Problem Statement: https://canvas.cmu.edu/courses/50537/assignments/903520

This repository contains notebooks for the three different tasks in assignment part A, as well as an additional notebook for the tasks in assignment part B. These notebooks document and contain all code used to interact with the specific models.

- creativeWriting_partA.ipynb is a Jupyter Notebook containing the code used to interact with OpenAI API models to accomplish a creative writing task for part A.

- The text summarization task in part A is completed using the interactive interface on OpenAI API server. Prompts and output have been documented in TextSummarizationPartA.doc.
